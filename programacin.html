🧠 Objetivo del programa

Este programa sirve para que un robot avance recto mientras verifica constantemente con sensores ultrasónicos si hay obstáculos delante.
Cuando detecta un obstáculo a menos de 42 cm de distancia, decide girar a la izquierda o derecha, dependiendo de cuál lado esté más libre.
Así, el robot da vueltas por el tablero evitando chocarse con paredes u objetos.

🧾 Bibliotecas utilizadas

#include <Servo.h>

#include <Ultrasonic.h>

#include <L298N.h>

Servo.h: para controlar un servo motor, que gira las ruedas como si fuera el volante del robot.


Ultrasonic.h: para manejar los sensores ultrasónicos, que miden la distancia a los objetos usando ondas de sonido que rebotan con las paredes y vuelven al sensor y gracias a esto calcula la distancia del objeto al sensor gracias a una fórmula matemática.


L298N.h: se usa para controlar motores con el driver L298N.

📦 Variables y pines

// Pines para los sensores ultrasónicos

#define TRIG_IZQ 8

#define ECHO_IZQ 9

Ultrasonic ultraIZQ(TRIG_IZQ, ECHO_IZQ,60000);


#define TRIG_CEN 6

#define ECHO_CEN 7

Ultrasonic ultraCEN(TRIG_CEN, ECHO_CEN,60000);


#define TRIG_DER 11

#define ECHO_DER 10

Ultrasonic ultraDER(TRIG_DER, ECHO_DER,60000);


Cada sensor está conectado a un par de pines TRIG/ECHO, y se usa la librería Ultrasonic para medir distancias. El número 60000 es un valor máximo de espera en microsegundos.


// Pines para driver L298N

int in2 = 4;

int in1 = 5;

int ena = 3;

Estos son los pines de control para el motor conectados al driver L298N:

in1 y in2 determinan la dirección del motor (adelante o atrás).
ena controla la velocidad con PWM.
int pinservo = 13;

Servo servo;

float centro=87.7;

El servomotor se conecta al pin 13 y centro es el ángulo que corresponde a las ruedas rectas que depende de la calibración del servo.

int MCentro;     //medición de ultrasonido del centro

int Mizq;     //medición de ultrasonido del centro

int Mdcha;     //medición de ultrasonido del centro

MCentro, Mizq, Mdcha: almacenan las distancias detectadas por cada sensor.

🔄 setup()

void setup() {

  pinMode(in2, OUTPUT);

  pinMode(in1, OUTPUT);

  pinMode(ena, OUTPUT);

  servo.attach(pinservo);

  servo.write(centro); 

  Serial.begin(9600);

}

Esta función se ejecuta una sola vez al iniciar el programa.

Configura los pines del motor como salidas para mandar la información hacia el motor para que la ejecute.

Conecta el servo al pin 13 y lo centra gracias a la variable “centro”.

Activa el puerto serie a unos 9600 watios .

♾️loop()

MCentro=ultraCEN.Ranging(CM);

Mizq=ultraIZQ.Ranging(CM);

Mdcha=ultraDER.Ranging(CM);

Se almacenan en variables ya creadas previamente para la medición de la distancia de los ultrasonidos, usando comandos de la libreria de Ultrasonic.


 if(MCentro <= 42){

   

    if (Mizq>Mdcha){

      izquierda(670) ;

      Serial.println("izquierda");

    }

    if (Mizq<Mdcha){

      derecha(660);

      Serial.println("derecha");

    } 

  }

 if(MCentro < 42){

   recto(50);

}

Aqui se utiliza un condicional para decir que si la distancia es menor o igual de 42, use otro condicional y sino, que siga recto gracias a otra condicional.

Si la distancia es inferior a 42 centimetros, empezaria otro condicional que su función seria comparar las distancias de izquierda y derecha y el que sea ayor de los dos es para el lado al que se gira gracias a un subprograma llamado izquierda, o derecha

↪️Subprogramas

Los subprogramas son fragmentos de código que realizan tareas específicas y suelen ser reutilizados en diferentes partes de un programa.


void recto(int tiempo){

  servo.write(centro);

  digitalWrite(in2, HIGH);

  digitalWrite(in1, LOW);

  analogWrite(ena, 200);

  delay(tiempo);

}

Este subprograma es el que hace que los motores se muevan rectos hacia delante mientras que el servo está parado justo mirando hacia el frente gracias a la librería L289N.


void izquierda(int tiempo){

  servo.write(45);

  digitalWrite(in2, HIGH);

  digitalWrite(in1, LOW);

  analogWrite(ena, 200);

  delay(tiempo);

  servo.write(centro);

}

void derecha(int tiempo){

  servo.write(135);

  digitalWrite(in2, HIGH);

  digitalWrite(in1, LOW);

  analogWrite(ena, 200);

  delay(tiempo);

  servo.write(centro);

}

void atras(int tiempo){

  servo.write(centro);

  digitalWrite(in2, LOW);

  digitalWrite(in1, HIGH);

  analogWrite(ena, 100);

}

void stop(int tiempo){

  servo.write(centro);

  digitalWrite(in2, LOW);

  digitalWrite(in1, LOW);

  analogWrite(ena, 0);

}

Estos subprogramas funcional igual que el primero pero se le cambian algunos parámetros para que en vez de que vaya hacia delante, pueda ir hacia atrás, derecha, izquierda o que esté parado.
